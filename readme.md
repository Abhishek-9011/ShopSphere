## Features

- Easy Shopping: Smooth product browsing and purchasing experience.
- Seller Panel: Simple onboarding and product listing for sellers.
- Dynamic Dashboard: Real-time tracking of sales, orders, and inventory.
- Secure Login: Protected authentication for users and sellers.
- Responsive UI: Works perfectly across all devices.

## Tech Stack

**Client:** React

**Server:** Node, Express

**Database:** MongoDb
